## Fleetlock client

Go implementation of `FleetLock` protocol.

### Example

```
$ fleetlockctl --help
Usage:
  fleetlockctl  [flags]

Flags:
  -h, --help   help for fleetlockctl
```

### Build

requirements:
  * `go` in the path

```
$ make
```

ref: https://coreos.github.io/zincati/development/fleetlock/protocol/
